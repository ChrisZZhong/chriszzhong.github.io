```java
public int[] shuffle() {
    int[] res = nums.clone();
    Random rd = new Random();
    for (int i = res.length - 1; i > 0; i--) {
        int j = rd.nextInt(i + 1);
        int tmp = res[i];
        res[i] = res[j];
        res[j] = tmp;
    }
    return res;
}
```
